# Contributing to Golden Palms CRM

Thank you for your interest in contributing to the Golden Palms CRM system built under Nexus Solutions!

## Development Setup

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/golden-palms-crm.git`
3. Install dependencies: `composer install`
4. Copy `.env.example` to `.env` and configure your environment
5. Set up your database (see README.md for details)
6. Create a new branch: `git checkout -b feature/your-feature-name`

## Code Style

- Follow PSR-12 coding standards
- Use type hints where possible
- Add PHPDoc comments for public methods
- Keep functions focused and single-purpose

## Commit Messages

Use clear, descriptive commit messages:
- `feat: Add new lead scoring feature`
- `fix: Resolve database connection issue`
- `docs: Update API documentation`
- `refactor: Improve email service structure`

## Pull Request Process

1. Ensure your code follows the project's coding standards
2. Update documentation as needed
3. Test your changes thoroughly
4. Submit a pull request with a clear description of changes

## Reporting Issues

When reporting issues, please include:
- PHP version
- Database version
- Steps to reproduce
- Expected vs actual behavior
- Error messages or logs (if applicable)

## Questions?

For questions or support, please contact the development team.
